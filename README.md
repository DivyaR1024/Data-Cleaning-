# Data Cleaning Excel Project

## Objective
Clean raw transaction data and prepare it for analysis.

## Steps Performed
- Removed duplicates
- Handled missing values
- Corrected price per unit using reference table
- Flagged ERROR and UNKNOWN values
- Calculated Total Spent
- Removed invalid transaction dates

## Files
- Raw_data.xlsx – original dataset
- cleaned_data.xlsx – cleaned final dataset
- Project overview.xlsx – Expalantion about Project
